# 安装文档双语完整性需求文档

**日期**: 2026-03-25
**任务类型**: 文档治理 - 安装文档双语补齐
**优先级**: 高
**执行模式**: 双语面对齐

---

## 🎯 目标（Goal）

检查 `docs/install` 下的安装文档双语完整性，补齐当前只有中文、缺少英文版本的文件，并修正英文入口页中仍然指向中文文件的链接。

## 📦 交付物（Deliverable）

1. 缺失的英文安装文档
2. 更新后的英文索引与入口链接
3. 一次可验证的双语完整性检查结果

## 🔒 约束条件（Constraints）

1. 保持现有安装口径不变
2. 英文文档应与中文文档在结构和事实层面对齐
3. 只补齐缺失语言版本，不随意扩写功能范围
4. README 与 `docs/install/*.en.md` 不应再链接回中文安装文档

## ✅ 验收标准（Acceptance Criteria）

1. `docs/install` 下所有公开安装说明都具备中英文版本，或明确属于语言无关文件
2. 当前缺失英文版的文件全部补齐
3. 英文入口页链接全部指向 `.en.md`
4. 通过脚本检查后，不再出现单语缺口

## 🚫 非目标（Non-Goals）

1. 不修改安装脚本
2. 不新增新的安装路径
3. 不重写整个安装文档体系
