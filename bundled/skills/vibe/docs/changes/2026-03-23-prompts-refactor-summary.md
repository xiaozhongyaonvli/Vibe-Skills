- 详细说明引用外部文档

## 📖 使用方法

### 用户如何使用新提示词

1. **选择合适的提示词**:
   - 安装全量版本 → `full-version-install.md`
   - 安装框架版本 → `framework-only-install.md`
   - 更新全量版本 → `full-version-update.md`
   - 更新框架版本 → `framework-only-update.md`

2. **复制提示词**:
   - 打开对应的 `.md` 文件
   - 复制 "复制给AI的提示词" 部分

3. **粘贴给AI**:
   - 在 Claude Code 或 Codex 中粘贴
   - 按照 AI 的提示回答问题
   - AI 会自动执行安装/更新

4. **查看详细文档**（可选）:
   - 如需了解规则详情 → 访问 `installation-rules.md`
   - 如需了解配置详情 → 访问 `configuration-guide.md`
