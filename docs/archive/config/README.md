# Archived Config Snapshots

This directory stores the small subset of historical machine-readable snapshots that still participate in current archive cross-links.

## Rule

- Most historical governance-board and wave-board snapshots now rely on git history rather than tracked archive files.
- Files kept here preserve historical machine-readable context only when another retained archive or audit surface still points at them.
- They must not be treated as live runtime, install, release, or test contracts.
